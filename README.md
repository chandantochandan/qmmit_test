# Chandan | Personal Profile

A minimalist, modern personal profile landing page built with HTML and Tailwind CSS.

## Features

- **Modern UI:** Features a sleek glassmorphism effect using Tailwind's backdrop-blur.
- **Responsive Design:** Optimized for all screen sizes, from mobile to desktop.
- **Vibrant Aesthetics:** Utilizes interactive gradients and smooth transitions.
- **Social Integration:** Includes placeholder links for GitHub, LinkedIn, and Twitter.

## Tech Stack

- **Frontend:** HTML5, Tailwind CSS
- **Styling:** Custom CSS Gradients & Tailwind utility classes

## Getting Started

To view the profile page:

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Open `index.html` in your favorite web browser.

## Project Structure

- `index.html`: The main entry point containing the structure and styles.
- `.qmmitignore`: Configuration for the `qmmit` tool.
- `.qmmit/`: Local logs and data for the `qmmit` tool.
